## /bin

This directory contains the **original, unmodified Audity 2000 OS binary images** used by the updater.

The files are provided **exactly as released**, with no alterations, patches, or reverse engineering applied.

### Contents

- audity-update-0.3-1.00.bin
- audity-update-0.3-2.00.bin
- audity-update-0.3-2.01.bin

---


Each binary corresponds to a specific Audity 2000 OS version and is intended solely for use with the Audity Update ROM.

These files are redistributed for **non-commercial, archival, and preservation purposes only**, in accordance with the original project license.
